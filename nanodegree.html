printscreen Jasmine Tests


In this project, I had to write tests for a web-based application that reads RSS feeds. To do this, I had to use the Jasmine framework. Jasmine is a behavior-driven development framework for testing JavaScript code. It does not depend on any other JavaScript frameworks. It does not require a DOM. And it has a clean, obvious syntax so that you can easily write tests.
